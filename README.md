Customer Churn Analysis - Telco Industry

Project Overview-
This project aims to identify the key factors that lead to customer attrition (churn) in a telecommunications company. By performing an End-to-End Exploratory Data Analysis (EDA), I’ve uncovered actionable insights to help the business improve retention strategies and identify high-risk customer segments.

Tech Stack & Tools-
Language: Python
Libraries: Pandas, NumPy, Matplotlib, Seaborn
Data Source: Telco Customer Churn (Kaggle)
Environment: Jupyter Notebook / VS Code

Key Insights from EDA-
* **Contract Type:** Customers on **Month-to-month** contracts have a significantly higher churn rate compared to those on one or two-year contracts.
* **Payment Method:** Users paying via **Electronic Check** show the highest tendency to leave.
* **Tenure:** New customers (0-6 months) are the most vulnerable to churning, suggesting a need for better onboarding.
* **Services:** Customers without **Online Security** and **Tech Support** services are more likely to churn.

Project Workflow-
1.Data Cleaning: Handled missing values (TotalCharges) and converted data types.
2.Feature Engineering: Categorized tenure into groups for better segmentation.
3.Visual Analysis: Used Seaborn/Matplotlib to create Heatmaps, Countplots, and Boxplots to find correlations.
4.Recommendations: Provided data-driven suggestions for marketing and loyalty programs.
